# MEAN-Stack-Expense-Tracker-Application
A full-stack Expense Tracker application built using the MEAN stack (MongoDB, Express.js, Angular, Node.js). Supports expense CRUD operations, real-time UI updates with Angular Signals, and a clean modular architecture.
Features include:

• Add, edit, delete, and view expenses  
• Angular 20 with Standalone Components + Signals  
• RESTful API built with Express & Mongoose  
• Centralized store using Angular Signals  
• Fully responsive UI  
• Real-time updates without page reloads  
• Clean folder structure and modular architecture  

Perfect for learning full-stack development using the latest Angular ecosystem.
